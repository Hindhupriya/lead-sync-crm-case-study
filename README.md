# lead-sync-crm-case-study
Product Design Case Study – LeadSync CRM (Lead Management Web Application)
LeadSync CRM – Lead Management System

# Overview

LeadSync CRM is a desktop web application designed to help car dealerships manage incoming leads efficiently. The system centralizes lead tracking, improves collaboration, and provides business insights through dashboards.

# Problem Statement

HSR Motors was managing leads using spreadsheets, which lacked real-time collaboration and analytics. This caused inefficiencies in follow-ups and performance tracking.
